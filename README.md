# Pendulum

## Setup

Clone the simulation repository and run the sync step below.

1. Clone the repo:

```bash
git clone https://github.com/Danglee1107/pendulum-simulation.git
```

2. Change into the project directory:

```bash
cd pendulum-simulation
```

3. Run the sync command:

```bash
uv sync
```

Note: I'm not certain what you mean by `uv sync` — if you meant `uvicorn` (for FastAPI apps), install requirements and run for development:

```bash
pip install -r requirements.txt
uvicorn main:app --reload
```
